## Project Description
A CSS button styles exercise featuring 4 button types with hover states.

## Technologies Used
- HTML5
- CSS3

## Features
- Primary button with solid background
- Secondary button with outline style
- Danger button in red
- Disabled button with grayed out state
- Hover states on all buttons

## How to Run
1. Clone this repository
2. Open `buttons.html` in your browser

## Lessons Learned
How to style buttons using CSS classes and hover states.
